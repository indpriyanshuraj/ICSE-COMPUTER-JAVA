## 🎯 Why This Project Exists

This repository provides organized Java practice programs aligned with the ICSE Computer Applications syllabus. This project provides a structured environment for practicing programming concepts learned in school and aims to:

1. Support learning by providing correct examples of common ICSE programs.
2. Build strong logic through practice in strings, arrays, loops, and conditionals.
3. Encourage clean coding with simple, readable Java programs.
4. Help students revise concepts through practical examples.

## 📂 Repository Structure

The repository is organized to make it easy to find programs by class, textbook chapter, topic, or practice source.

## Root Folders

- `class-8/` – Programs and practice material for Class 8.
- `class-9/` – Programs and practice material for Class 9.
- `class-10/` – Programs and practice material for Class 10.
- `programs/` – Topic-wise Java programs that are independent of any specific class or textbook.

## Class Folder Structure

Each class folder contains two main sections:

#### 📘 Books

Programs are organized according to commonly used ICSE textbooks.

Structure:
```text
class-(number)/
└── book/
    └── Book Name (Publication Year)/
        └── Chapter/
```
This organization allows students to quickly locate programs corresponding to a particular textbook chapter.

#### 📝 Questions

Additional practice questions that are not tied to a specific textbook are stored separately.

Structure:
```text
class-(number)/
└── question/
```
These may include:

- School examination questions
- Sample or model paper questions
- Online practice questions
- Custom practice questions created for revision

---

### Topic-wise Programs

The `programs/` directory groups programs by programming topic rather than by class. Each topic contains related examples and practice programs.

Structure:
```text
programs/
└── Topic/
    ├── Program 1
    ├── Program 2
    └── ...
```
This organization is useful for focused practice and revision of specific Java concepts regardless of class level.

## ⚙️ How to Use This Code

### Step 1: Get the Code

Option A: Download ZIP
1. Click the green "Code" button at the top of the page.
2. Select "Download ZIP".
3. Extract the downloaded folder.

Option B: Clone with Git
```sh
git clone https://github.com/indpriyanshuraj/ICSE-COMPUTER-JAVA.git
```

---

### Step 2: Installing Java and Setting Up the Environment

#### Use BlueJ (Recommended for ICSE students)

Download BlueJ from the official website:
- [www.bluej.org](https://www.bluej.org/)

Then:
1. Open BlueJ.
2. Click "Project" → "Open Project..."
3. Open the cloned or extracted project folder.
4. Double-click the class file.
5. Click Run.

#### Windows
Download and install Java from the official site:
- [www.oracle.com/java](https://www.oracle.com/java/)

Then open Command Prompt and run:
```cmd
cd path\to\your\folder
javac ProgramName.java
java ProgramName
```

#### Linux (Ubuntu/Debian)
```sh
sudo apt update
sudo apt install openjdk-17-jdk
```

Then run:
```sh
cd path/to/your/folder
javac ProgramName.java
java ProgramName
```

#### Fedora / CentOS
```sh
sudo dnf install java-17-openjdk
```

Then run:
```sh
cd path/to/your/folder
javac ProgramName.java
java ProgramName
```

#### macOS
Install Homebrew from the official site:
- [brew.sh](https://brew.sh/)

Then install Java:
```sh
brew install openjdk@17
```

Then run:
```sh
cd path/to/your/folder
javac ProgramName.java
java ProgramName
```

#### Termux (Android)
```sh
pkg update && pkg upgrade
pkg install git openjdk-17

git clone https://github.com/indpriyanshuraj/ICSE-COMPUTER-JAVA.git
cd ICSE-COMPUTER-JAVA
```

Then run:
```sh
cd path/to/your/program
javac ProgramName.java
java ProgramName
```
