# Security Policy

## Supported Versions

This is a personal educational repository containing ICSE practice programs.
It is not a production software project and does not have versioned releases.

## Reporting a Vulnerability

Although this repository contains only practice code, responsible disclosure
is still appreciated.

If you discover a security concern, please report it via:

- **Open an Issue:** https://github.com/indpriyanshuraj/ICSE-COMPUTER-JAVA/issues
- **GitHub Discussions:** https://github.com/indpriyanshuraj/ICSE-COMPUTER-JAVA/discussions

## Scope

Since this repository is for educational purposes only, typical security
vulnerabilities (e.g. authentication flaws, data breaches) do not apply.
Concerns worth reporting include:

- Malicious code accidentally introduced
- Dependency vulnerabilities (if any dependencies are added in future)
- Sensitive information accidentally committed (e.g. credentials)
